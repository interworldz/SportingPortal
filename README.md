\# Спортивний Портал України



Навчальний проект для РГР (розрахунково-графічної роботи), спрямований на створення інформаційного ресурсу для висвітлення спортивних подій.



\## Опис проекту

Портал забезпечує користувачів актуальною інформацією про спортивне життя України. Сайт реалізований за архітектурою клієнт-сервер:

\* \*\*Frontend:\*\* HTML5, CSS3, JavaScript (Bootstrap 5).

\* \*\*Backend:\*\* Node.js, Express.js.

\* \*\*API:\*\* Використовується для динамічного завантаження даних (новини, меню).



\## Структура проекту

\- `index.html` — головна сторінка з каруселлю та списком новин.

\- `about.html` — сторінка "Про нас".

\- `gallery.html` — галерея спортивних подій.

\- `contact.html` — сторінка з контактною інформацією та картою.

\- `news-detail.html` — сторінка перегляду окремої новини.

\- `server.js` — серверна частина на Node.js для обробки запитів.



\## Як запустити проект



1\. Переконайтеся, що у вас встановлено \[Node.js](https://nodejs.org/).

2\. Встановіть необхідні залежності:

&#x20;  ```bash

&#x20;  npm install express cors
3. Запустіть сервер:
	node server.js
4. Відкрийте index.html у вашому браузері (через Live Server або напряму).
