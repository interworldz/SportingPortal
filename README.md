# Спортивний Портал України

Навчальний клієнт-серверний проект для висвітлення спортивних подій України. Сайт містить головну сторінку з каруселлю, окрему стрічку новин з фільтрами та пагінацією, детальні сторінки матеріалів, галерею, інформаційну сторінку та контакти.

## Технології

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5.
- **Backend:** Node.js, Express.js.
- **API:** JSON-ендпоїнти для отримання новин і меню.
- **Зображення:** локальні файли з папки `images/`.

## Функціонал

- Динамічна карусель на головній сторінці.
- Окрема сторінка `news.html` зі списком усіх новин.
- Фільтрація новин за категоріями: футбол, баскетбол, легка атлетика, бокс.
- Пагінація по 3 новини на сторінку.
- Детальна сторінка новини `news-detail.html?id=...`.
- Відображення фото, дати, категорії, повного тексту та джерела новини.
- Галерея локальних спортивних зображень.
- Сторінка контактів з інтегрованою картою.

## Структура проекту

- `index.html` — головна сторінка з описом порталу, каруселлю та переходами до розділів.
- `news.html` — сторінка зі списком новин, фільтрами та пагінацією.
- `news-detail.html` — сторінка перегляду окремої новини.
- `about.html` — сторінка з описом проекту.
- `gallery.html` — галерея спортивних фото.
- `contact.html` — сторінка контактів.
- `server.js` — Express-сервер і API з даними новин.
- `css/styles.css` — усі стилі проекту.
- `js/index.js` — логіка каруселі та лічильника на головній.
- `js/news.js` — логіка списку новин, фільтрів і пагінації.
- `js/news-detail.js` — логіка детальної сторінки новини.
- `images/` — локальні зображення для новин, галереї та каруселі.

## API

### `GET /api/news`

Повертає масив усіх новин.

### `GET /api/news/:id`

Повертає одну новину за її індексом. Якщо новину не знайдено, сервер повертає `404`.

### `GET /api/menu`

Повертає пункти навігаційного меню.

## Як запустити

1. Встановіть Node.js.
2. Встановіть залежності:

```bash
npm install
```

3. Запустіть сервер:

```bash
npm start
```

4. Відкрийте сайт у браузері:

```text
http://localhost:3000
```

## Перевірка

Для базової перевірки синтаксису сервера:

```bash
npm test
```
